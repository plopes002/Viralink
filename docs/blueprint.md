# **App Name**: ViralinkAI

## Core Features:

- Social Account Monitoring: Monitor connected Instagram, Facebook, and WhatsApp Business accounts for key metrics and user engagement.
- AI-Powered Post Generation: Utilize Gemini 1.5 / Vertex AI tool to generate engaging social media posts based on a specified theme and desired tone. Incorporate relevant hashtags and attention-grabbing titles.
- Automated Content Scheduling: Schedule generated and manually created content to be posted automatically at optimal times, ensuring consistent engagement.
- Competitor Analysis: Track competitor performance on social media, analyzing their posting frequency, engagement rates, and content themes to identify opportunities for improvement. Utilizes Firestore.
- Comprehensive Reporting: Generate detailed reports on social media performance, providing insights into audience demographics, engagement metrics, and content effectiveness.
- Multi-Tier Subscription Plans: Offer tiered subscription plans (Basic, Pro, Expert) with varying features and usage limits, catering to different user needs and budgets, managed via Stripe.
- Automated Message Responses: Set up rules to automatically respond to messages, or have the system learn the right time to trigger and inject rules, automating tasks such as converting a DM to a Whatsapp.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) evokes a sense of trust and reliability, vital for a social media management platform.
- Background color: A light blue (#E1F5FE) offers a clean, modern look, providing ample contrast for text and visual elements.
- Accent color: A lively green (#90EE90) highlights calls to action and important metrics, promoting engagement and user confidence.
- Headline font: 'Space Grotesk' sans-serif provides a techy, contemporary look for headlines and titles.
- Body font: 'Inter' sans-serif provides excellent readability in user interface.
- Use a consistent set of modern, minimalist icons throughout the platform to represent different features and functionalities.
- Incorporate subtle animations and transitions to enhance user experience and provide feedback on interactions.

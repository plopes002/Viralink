# ViraMind

Plataforma de automação e gestão de interações para redes sociais.

## 📌 Sobre o projeto

O ViraMind permite que empresas conectem suas contas do Instagram, Facebook e WhatsApp para:

- Gerenciar mensagens e comentários em um único painel
- Automatizar respostas com base em regras
- Monitorar engajamento e interações
- Analisar desempenho de conteúdos
- Acompanhar concorrentes

## 🚀 Funcionalidades principais

- 🔗 Conexão com Instagram e Facebook (API oficial Meta)
- 💬 Centralização de mensagens e comentários
- ⚙️ Motor de automação (respostas automáticas)
- 📊 Dashboard com métricas de engajamento
- 👥 Gestão de múltiplas contas e perfis
- 🧠 Análise de interações e leads

## 🏗️ Tecnologias utilizadas

- Next.js
- Firebase (Firestore, Auth, Hosting)
- Meta Graph API (Instagram e Facebook)
- Node.js
